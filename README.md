## ğŸ¯ Abstract
The **Advanced Sorting Algorithm Visualizer** is a GUI-based Python project that demonstrates and compares the working of multiple sorting algorithms in real-time.  
Developed using **Tkinter** for the graphical interface and **Matplotlib** for data visualization, this project aims to provide students with a clear and interactive way to understand the core concepts of **Design and Analysis of Algorithms (DAA)**.

Users can:
- Select a sorting algorithm
- Control sorting speed and data size
- Watch the algorithm sort data step-by-step with dynamic animations
